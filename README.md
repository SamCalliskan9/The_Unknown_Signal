**Project Overwiew

"The Lost Signal" is a small interactive console game written in Python.
The player takes the role of **Alex Carter**, a young engineer on a remote research station who receives a mysterious signal from a unknown source.
The goal of the game is to **Decode the signal** and survive the challenges while exploring the story.

---

- Story-driven narrative with interactive elements.
- Frequency guessing mini-game (1,5) with limited attempts.
- Good ending and bad ending based on player's decisions.
- Simple text-based interface using Python's console.
- Character name input ('add_me()' function) to personalize the story.
